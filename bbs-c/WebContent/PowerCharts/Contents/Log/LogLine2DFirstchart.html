Creating a simple Log Line chart

To begin with, let us create a simple chart with the same data that we used to create the Logarithmic Column 2D chart. The chart will look as under:

And the data for same can be listed as under:
  • XML
  • JSON
<chart caption='Business Results 2005 v 2006' xAxisName='Month' yAxisName='Revenue' numberPrefix='$' formatNumberScale='1' canvasPadding='25'>
   <categories>
      <category label='Jan' />
      <category label='Feb' />
      <category label='Mar' />
      <category label='Apr' />
</categories>
 <dataset seriesName='2006'>
      <set value='47400' />
      <set value='39800'/>
      <set value='55800' />
      <set value='36800' />
</dataset>
<dataset seriesName='2005'>
      <set value='28000'/>
      <set value='31500'/>
      <set value='42500'/>
      <set value='65000'/>
</dataset>
</chart>

In the above XML we have done the following:

  • Created the <chart> element which is the root element of the chart
  • Defined the caption of the chart using the caption attribute
  • Defined the label of the x-axis and the y-axis using the xAxisName and yAxisName attributes
  • Defined the prefix of the numbers using the numberPrefix attribute
  • Set scaled number using numberFormatScale='1'
  • Set the distance between the chart canvas and the dataplots using the canvasPadding attribute
  • Defined years as x-axis labels using <category> elements
  • Created two <dataset> elements (2006 and2005) to provide data
  • Provided data using the value attribute in each of the <set> elements
