Inverse y-Axis Charts

The Inverse y-axis charts are similar to the normal FusionCharts XT charts but for the fact that y-axis in these charts are inversed. Shown below is a normal FusionCharts XT y-axis and the Inverse chart y-axis:

Normal FusionCharts XT y-axis Inverse y-axis
 

As you can see in the above images, in the Inverse y-axis charts, the chart lower limit is shown at top and the upper limit is shown at bottom of chart canvas. This makes the chart very effective to plot information like ranking, hits trend etc.

Currently, we have the following 3 inverse axis charts available:

  1. Inverse y-axis Area Chart
  2. Inverse y-axis Column Chart
  3. Inverse y-axis Line Chart

All these charts extend from the original FusionCharts v3 charts and hence the feature set is same. Only the axis has been reversed.
