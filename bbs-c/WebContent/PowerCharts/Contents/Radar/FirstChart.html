Creating a Simple Radar Chart

Here, we will create a simple radar chart which allows a meaningful comparison of two different cars based on several entities such as speed, engine power, cost etc.

The chart would look something as under:

The XML/JSON for the above chart is listed below:

  • XML
  • JSON
<chart caption='Car Comparison' >
   
   <categories>
      <category label='Speed' />
      <category label='Engine Power' />
      <category label='Comfort' />
      <category label='Cost' />
      <category label='Safety' />
   </categories>

   <dataset seriesName='Car Y'>
      <set value='70' />
      <set value='80' />
      <set value='47' />
      <set value='76' />
      <set value='69' />
   </dataset>

   <dataset seriesName='Car Z'>
      <set value='90' />
      <set value='65' />
      <set value='60' />
      <set value='65' />
      <set value='45' />
   </dataset> 
  
</chart>      
{

  },
  "categories":[{
      "category":[{

        },

          "label":"Safety"
        }
      ]
    }
  ],
  "dataset":[{

          "value":"65"
        },

          "value":"60"
        },
        {
          "value":"65"
        },
        {
          "value":"45"
        }
      ]
    }
  ]
}

In the data above, we have:

  • Created the <chart> element and specified a caption to the chart. You can define various other functional and cosmetic properties for the <chart> element which has been discussed in the Chart XML Sheet.
  • Added the <categories> element, in which a custom label for each <category> element has been defined
  • Used two <dataset> elements to contain the data
  • Provided data for the respective series using <set> elements.
