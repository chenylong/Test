XML/JSON Attributes for Legend

The series name of each data-set shows up in the legend of the chart. This helps you relate a data plot to its series name using its color. The legend does not appear for drag-node, waterfall, single series spline, single series spline-area, kagi and multi-level pie charts .

Notice the legend below the chart which helps identify the series name. Starting v3.2, each series is represented by its own icon.

Starting v3.2, the legend is interactive. That is, when you click each legend icon, it hides the data set corresponding to that item. In the example below, 'Current Year' icon was clicked and this resulted in the chart shown below. Note how the 'Current Year' icon has also been converted to gray-scale. To show the area back, you click this icon again.

Legend Customization Options

PowerCharts XT offers a lot of customization options for legend - both functional and cosmetic. Let's quickly run through each of them.
Setting legend position to right

The legend can be placed in two positions: RIGHT and BOTTOM. It gets placed below the canvas by default. To place it to the right of the chart canvas, set legendPosition='RIGHT'.

<chart legendPosition='RIGHT'...>

This results in a chart as under:
Configuring legend icon size

You can increase the size of legend icons by specifying a value between 0 and 5 for the attribute legendIconScale, that is, a value greater than 0 (not equal to) and less than 5. Shown below is an example of legendIconScale='2'. The chart will automatically shrink to adjust the legend.
Legend Scroll bar

If you have a lot of items in legend (that is, a lot of data-series) and not enough space on the chart, the legend automatically shows up the scroll bar. Shown below is an example:

You can customize the visual properties of this scroll bar using following attributes:
Attribute Name Range Description
legendScrollBgColor Hex Code This attribute lets you configure the background color of the scroll bar.
legendScrollBarColor Hex Code This attribute lets you configure the bar color of the scroll bar.
legendScrollBtnColor Hex Code This attribute lets you configure the color of buttons of the scroll bar.
Setting number of columns in legend (when rendering at bottom of chart)

When rendering the legend at bottom of chart, you can also customize the number of columns across which the legend items will be split. If PowerCharts XT can accommodate the items within those number of columns, it will render accordingly. This can be done by setting:

<chart legendNumColumns='2' ...>

This results in a chart as under. Note, that by default the chart at this size would have rendered the legend in a single column. Setting two columns forces it to wrap first label. However, if you specify number of columns as three, PowerCharts XT will ignore it, as there is not enough space to render three columns horizontally.

Note that this attribute does not work when rendering legend at the right side of chart.

Reversing the order of items in legend

The order of items in legend can be reversed by setting the XML/JSON as under.

Hiding the legend

The legend can be hidden as well using showLegend='0'.

<chart showLegend='0'...>
Customizing the legend

The looks of the legend can be customized using the following attributes:
Attribute Name Range Description
legendBgColor Hex Code Background color for the legend.
legendBgAlpha 0-100 Background alpha for the legend.
legendBorderColor Hex Code Border Color for the legend.
legendBorderThickness In Pixels Border thickness for the legend.
legendBorderAlpha 0-100 Border alpha for the legend.
legendShadow 0/1 Whether to show a shadow for legend.
legendScrollBgColor Hex Code If you have too many items on the legend, a scroll bar shows up on the same. This attribute lets you configure the background color of the scroll bar.
legendScrollBarColor Hex Code If you have too many items on the legend, a scroll bar shows up on the same. This attribute lets you configure the bar color of the scroll bar.
legendScrollBtnColor Hex Code If you have too many items on the legend, a scroll bar shows up on the same. This attribute lets you configure the color of buttons of the scroll bar.
Using Styles to customize the font properties

Using Styles, the font properties of the legend can be customized.

  • XML
  • JSON
<chart...>
    ...
    <styles>
       <definition>
           <style name='myLegendFont' type='font' size='12' color='078BC5' bold='1' />
       </definition>
       <application>
           <apply toObject='Legend' styles='myLegendFont' />
       </application>
    </styles>  
</chart>
  {
  "chart":{...},
  ...
  "styles":{
    "definition":[{
        "name":"myLegendFont",
        "type":"font",
        "size":"12",
        "color":"078BC5",
        "bold":"1"
      }
    ],
    "application":[{
        "toobject":"Legend",
        "styles":"myLegendFont"
      }
    ]
  }
}

To read more on Styles, please refer to PowerCharts XT and Styles.
Animation using Styles

The legend can be animated using Styles. Here, we will give a fade-in effect to the legend.

  • XML
  • JSON
<chart ... >
        ...
          <styles>
             <definition>
                  <style name='myAnim' type='animation' param='_alpha' start='0' duration='1'/>
             </definition>
             <application>
                  <apply toObject='Legend' styles='myAnim' />
             </application>
          </styles>
</chart>
{
  "chart":{...},
...
  "styles":{
    "definition":[{
        "name":"myAnim",
        "type":"animation",
        "param":"_alpha",
        "start":"0",
        "duration":"1"
      }
    ],
    "application":[{
        "toobject":"Legend",
        "styles":"myAnim"
      }
    ]
  }
