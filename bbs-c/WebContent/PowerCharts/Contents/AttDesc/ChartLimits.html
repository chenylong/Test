XML/JSON Attributes for Chart Limits

Chart limits are the minimum and maximum y-axis values. PowerCharts XT automatically calculates the best chart limits when you do not specify them explicitly.

In PowerCharts XT, the axis limits encompasses all values present in the chart, that is, no data value can be below the minimum y-axis value or above the maximum y-axis value - they have to be within the axis range. Even if you force an axis limit that is lower or higher than this, PowerCharts XT will automatically adjust it.

The XML/JSON code going into the above is:

  • XML
  • JSON
<chart caption='Monthly Revenue' xAxisName='Month' yAxisName='Revenue' numberPrefix='$' showValues='0'>
     ....
</chart>
{
  "chart":{
    "caption":"Monthly Revenue",
    "xaxisname":"Month",
    "yaxisname":"Revenue",
    "numberprefix":"$",
    "showvalues":"0"
  },
...
}

Note that the chart limits have not been defined.
Setting chart limits explicitly

You can set the chart limits explicitly using the yAxisMaxValue and yAxisMinValue attributes.

<chart yAxisMinValue='0' yAxisMaxValue='100' ...>
 
Set Adaptive y-Min

You can set whether the y-axis lower limit would be 0 (in case of all positive values on chart) or should the y-axis lower limit adapt itself to a different figure based on values provided to the chart.

  • XML
  • JSON
<chart caption='Monthly Revenue' xAxisName='Month' yAxisName='Revenue' numberPrefix='$' showValues='0' setAdaptiveYMin='1' >
      ....
</chart>
{
  "chart":{
    "caption":"Monthly Revenue",
    "xaxisname":"Month",
    "yaxisname":"Revenue",
    "numberprefix":"$",
    "showvalues":"0",
    "setAdaptiveYMin":"1"
  },
  ...
}

Hiding limits

You can always hide the limit values of a chart. To hide the limits you need to set the following: 

<chart showLimits='0' ..>

Note: Chart limits cannot be hidden for Drag-able charts.
