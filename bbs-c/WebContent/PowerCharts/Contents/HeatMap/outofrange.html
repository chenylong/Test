Handling out of range data

If the data values are beyond the numeric ranges defined in the <colorRange> element the chart will show blank dataplots. For example, the numeric range defined is 0-30, 30-70 and 70-100 and in the <set> elements there are values such as -10 and 123. The chart will display all the data values within the range but will show blank dataplots in place of -10 and 123. The image below shows a chart with two blank dataplots:

To include all the values provided in the data, you can set the mapByPercent attribute to '1' in the <colorRange> element. This uses percentage mapping and displays the out of range values. This also shows the values in percentage. So, it considers the value - 10 as 0% and the value 123 as 100%. When you hover the mouse cursor on a particular dataplot the tool tip displays the actual value. Note that to convert the data values in percentage, you need to create color ranges with 0 as the lower limit and 100 as the upper limit. In category based heat map chart, if you provide a category name by the attribute colorRangeLabel through the <set> element which is not defined in the <colorRange> element, the chart will show blank dataplot.
