Outliers

Outliers are those values which are numerically distant from the set of numbers provided. Outliers are indicative values which may occur by chance or by error for any set of data provided. Outliers are not included in the statistical calculation of the five summaries. Rather these are placed as separate points on the chart. To show outliers for a set of data you need to specify the outlier values using the outliers attribute through the <set> elements for each set of data.

In the above screenshot, the chart is displaying the outlier values by the triangle icons of each dataplot. This icon is customizable about which we will discuss later.

The data for the above chart is given below:

  • JSON
<chart caption='Visits to a Website' subCaption='In three years' showValues='0'>
...
<dataset seriesName="January" lowerBoxColor='019FAA ' upperboxColor='D4DFFF' mediancolor='FF0000' >

In the above data, the outliers attribute is used to provide the outlier values through the <set> element. So, all the dataplots are displaying the outliers value icon.

The attributes used to customize the outlier icon are given below:
Attribute Name Type Range Description
outlierIconShape String Specifies the shape of the icon. There are two options - polygon and spoke.
outlierIconRadius Number Specifies the radius of the outlier icon.
outlierIconSides Number Specifies the number of sides for the icon. If the outlierIconShape is set as polygon, then the shape of the icon will depend on the number of sides. For example, 3 sides will display a triangle, 8 will display a circle.
outlierIconColor Hex code Sets the color of the outlier icon.
outlierIconAlpha Number 0-100 Sets the transparency of the outlier icon.
 
Show all Outliers

In the above example, all the outliers provided in each <set> element appear on the chart since they reside in the range of the y-axis scale. There can be instances where the outlier values are beyond the range of the y-axis scale. In this situation, the outlier values are not displayed to maintain the visual integrity of the dataplots. A sample chart is shown below:

In the above screen shot, only one outlier value is displayed though all the <set> elements have outlier values. The data for the above chart looks as under:

  • XML
  • JSON
<chart caption='Visits to a Website' subCaption='In three years'>
...
<dataset seriesName="January" lowerBoxColor='019FAA ' upperboxColor='D4DFFF' mediancolor='FF0000' >
    <set value='60,133,35,67,89,137,259' outliers='290'/>
    <set value='139,197,175,114,39,67,191' outliers='350'/>
    <set value='200,107,197,239,53,26,97' outliers='-30, -90'/>
</dataset>
</chart>
{
  "chart":{
    "caption":"Visits to a Website",
    "subcaption":"In three years"
  },
  ...
  "dataset":[{
      "seriesname":"January",
      "lowerboxcolor":"019FAA ",
      "upperboxcolor":"D4DFFF",
      "mediancolor":"FF0000",
      "data":[{
          "value":"60,133,35,67,89,137,259",
          "outliers":"290"
        },
        {
          "value":"139,197,175,114,39,67,191",
          "outliers":"350"
        },
        {
          "value":"200,107,197,239,53,26,97",
          "outliers":"-30, -90"
        }
      ]
    }
  ]
}

In the above data, there is only one outlier value which resides within the range of the y-axis scale. So, except that value all the other outliers are not displayed in the chart.

To display all the outlier values including the values which are beyond the range of the y-axis scale, set the showAllOutliers attribute to '1' in the <chart> element. A sample chart with all the outlier values displayed is given below:

In the above screen shot, the chart is displaying all the outliers. The data will be as under:
