Download Package Contents

PowerCharts XT Download Package contains the following items. Please note that the availability of the items depends on the type of license purchased.
FusionCharts JavaScript Classes
 

FusionCharts JavaScript Class is present in Download Package > Charts folder. FusionCharts JavaScript Class for PowerCharts XT comprises of four JavaScript files - FusionCharts.js, FusionCharts.HC.js, FusionCharts.HC.PowerCharts.js and jquery.min.js. FusionCharts.js is the core module of FusionCharts JavaScript Class that helps you embed charts in your web pages in a user-friendly way and offers functionalities like updating chart data, retrieving chart data, supporting multiple data formats, event handling etc. Additionally, it also helps in rendering JavaScript fallback charts (using FusionCharts.HC.js, FusionCharts.HC.PowerCharts.js and jquery.min.js). The FusionChartsExportComponent.js allows control over the Flash Client-side Export Component and provides interfaces to link the Flash Client-side Export Component to the charts present in the web page.

If you are already using FusionCharts XT in your web application, you can safely replace your existing FusionCharts.js and other JavaScript files with the latest files which you have received in the PowerCharts XT pack. The latest FusionCharts JavaScript Classes are compatible with your existing charts and JavaScript API.

The legacy FusionCharts JavaScript Class (from v3.1.1) is also provided in Download Pack > SourceCode > JavaScript > Legacy folder.
SWF Files (Ready to use charts)
 

PowerCharts XT is essentially a bundle of JavaScript and SWF files. These files are present in Download Package > Charts folder. Whenever you need to create charts for a new web application, just copy these files and paste it in your application. Additionally, the Flash Client-side export component, which comes as a separate SWF file (FCExporter.swf) is also present in Charts folder.
Sample Codes
 

Various basic implementation samples are present in Download Package > Code folder. These are ready-to-use code samples, which you can copy-paste and run. Just make sure that you copy the charts (from Charts folder) along with the respective example, else you would get an empty placeholder instead of the chart.

Server Side Export Handlers (Image/PDF Saving Scripts)
 

The scripts which allow you to save PowerCharts XT as PDF/images at Server-side are present in Download Package > ExportHandlers folder. Please refer to "Exporting as Image/PDF " section in documentation on how to use them.

Note: Starting PowerCharts XT v3.3.0, JavaScript charts can be exported and saved to a server disk. To do so, you need to configure your own server. The sever should support PHP and JAVA (1.3 or above). Also, you need to place the FusionCharts PHP export handler and Apache Batik SVG Rasterizer library in your server.
Tools
 

There are a few tools that user can use to build and configure charts in Download Package > Tools folder.

  • FusionCharts Data Format Conversion Tool : This is a simple tool that lets you convert PowerCharts XT XML to JSON and vice versa. This tool is present in the FCDataConverter folder.
  • Flash Player Security Setup : Script files present in the FlashPlayerSecuritySetup folder to configure Flash Player Global Security settings - offline.

Chart XSD schema
 

The XSD schemas of all the charts are present  in Code > ChartXSD folder.
Chart Source Code
 

f you have the source code license of PowerCharts XT (as part of Enterprise, Enterprise Plus or Redistributable license), you can find the source code at Download Package > SourceCode folder.

The source code of Flash charts are present in Download Package > SourceCode > Flash folder. This folder contains all the .fla files. All the ActionScript files are under the package com.fusioncharts.

The source code of JavaScript charts and FusionCharts JavaScript Classes are present in Download Package > SourceCode > JavaScript folder.
Chart Samples
 

We have built a few chart samples for your viewing pleasure. These are present in Download Package > Gallery folder.
